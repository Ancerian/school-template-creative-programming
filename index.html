Біблотека:

  • TF novel

    Бета версія проекту

  • TF novel p5js port

    Цей проєкт є інтерактивною історією, створеною за допомогою бібліотеки p5.js. Користувач може взаємодіяти з різними сценами, обираючи варіанти дій через кнопки. Фон кожної сцени змінюється відповідно до сюжету, а інтерфейс адаптується до розміру вікна браузера.

  • технологии

    Перелік використаних технологій: 1.p5.js - бібліотека для створення інтерактивної графіки та анімацій. 2. JavaScript - основна мова програмування для реалізації логіки проєкту. 3. HTML/CSS - для базового оформлення сторінки (наприклад, стилі для `body` та `canvas`). 4. Браузерний API - використовується для обробки подій, таких як `mousemove` та `resize`. 5. Консоль браузера - для відстеження помилок і завантаження ресурсів через `console.log` та `console.error`.
